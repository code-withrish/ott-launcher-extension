# OTT Quick Launcher

A simple Chrome extension to open OTT apps like Netflix, Hotstar, Zee5, and Prime Video with one click.

## Installation
1. Open `chrome://extensions` in Chrome.
2. Enable Developer Mode.
3. Click "Load Unpacked" and select the `ott-launcher` folder.
4. The extension will appear in your toolbar.

## Future Scope
